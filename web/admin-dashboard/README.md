# Admin Dashboard

React web application for platform administrators to manage the Flow Cart e-commerce platform.

## Features

- Platform overview dashboard
- User management
- Merchant management & approval
- Product moderation
- Global order monitoring
- Platform analytics

## Setup

```bash
npm install
npm run dev
```

The dashboard will be available at http://localhost:3001

## Tech Stack

- React 18
- TypeScript
- Material-UI
- Material-UI DataGrid
- Zustand (state management)
- React Router
- Vite
- Recharts (analytics)

## Admin Features

- **Dashboard**: Platform-wide statistics and activity
- **Users**: Manage all users, activate/deactivate accounts
- **Merchants**: Approve merchant applications, manage stores
- **Products**: Moderate product listings
- **Orders**: Monitor all orders across the platform
- **Analytics**: Platform performance metrics and insights
